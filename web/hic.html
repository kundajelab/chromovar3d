Methods

Data processing

We processed HiC data using the HiCup pipeline (http://www.bioinformatics.babraham.ac.uk/projects/hicup/), which performs mapping of the data to an in-silico digested (HindIII) reference genome hg19, filtering for common artifacts, such as di-tag where both reads map onto the same restriction fragment as well as outputs quality measures, such as the fraction of intra-chromosomal to inter-chromosomal read pairs. Generally, the higher the fraction of intra-chromosomal read pairs, the higher the quality of the HiC data, since it is assumed that most interactions happen within one chromosome. With over 65% intrachromosomal read pairs our data is of very good quality.

Estimating physical proximity

One major challenge when analyzing HiC data is that the number of potential interactions is orders of magnitudes larger than the number of sequencing reads obtained in the experiment, even when the library is sequenced to the depth reported in this study. However, assuming that if two sites that largely interact with the same set of sites are likely to also interact with each other, we employ a co-variance measure to estimate the physical proximity between two fragments. Such a measure is particularly useful in our study, where we integrate HiC data with genetic associations to identify long-distance genetic associations that are supported by physical interaction of the DNA. Since we can employ genetic association as an orthogonal filter to HiC interaction calls we are mainly interested in minimizing the number of false negative interactions calls and are less concerned with false positive ones.

Our HiC-interaction analysis is based on restriction-fragment level resolution. We obtained the interaction count for each pair of fragments. To estimate the proximity between two restriction fragments A and B we calculated their co-variance as the fraction of fragments that interact with both A and B, normalized by the number of fragments interacting only with A or only with B. The distribution of the so obtained correlations is shown in Fig. S2A. As expected this measure has strong distance dependence with fragments in close genomic proximity showing very high correlation scores Fig. S2B.
